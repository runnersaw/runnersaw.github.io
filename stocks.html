Stocks!

This product was the final project for my Software Design class in my second semester. Along with Griffin Tschurwald, Greg Coleman, Kevin Suzuki, and Maor Bernstein, we created a web app that would predict stock prices. This web app, created with Flask and Python, databased stocks with SQL, pulled tweets from Twitter with the Pattern module, and analyzed the stock's prices using multivariable linear ridge regression.

You can find the source code on GitHub. The algorithm won't run unless you install PostgreSQL, create a stocks database, and run the stockMaker.py file to populate the database.
